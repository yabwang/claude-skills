# AI 博客更新器

自动更新博客的 AI 最新进展文章。

## 工作流程

1. **获取最新 AI 新闻** - 使用 WebSearch 搜索近期 AI 领域重要动态
2. **整理关键内容** - 提取重要新闻、新模型发布、工具更新等
3. **生成文章内容** - 按模板结构生成 markdown 文章
4. **更新博客文件** - 更新 `docs/ai/latest-developments.md`
5. **Git 提交推送** - 提交更改并推送到远程仓库

## 输出格式

文章结构：
- 🚀 **重磅发布**: 新模型、新工具发布
- 💡 **技术突破**: 重要技术进展
- 🛠️ **工具更新**: 开发工具更新
- 📚 **学习资源**: 新教程、文档

## 参数

- `$ARGUMENTS`: 可选的主题关键词，如"Claude"、"OpenAI"等

## 注意事项

- 每次更新保留原有内容，追加新动态
- 更新最后修改时间
- 使用中文撰写
